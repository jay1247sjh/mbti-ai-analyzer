# MBTI AI Analyzer

A full-stack MBTI questionnaire app built with React + TypeScript + Tailwind CSS and Node.js + Express.

## Run

```bash
npm install
npm run dev
```

Or use one command redeploy script:

```bash
npm run redeploy
```

Default npm mirror is `https://registry.npmmirror.com/` (configured in project `.npmrc`).
You can override it temporarily:

```bash
NPM_REGISTRY=https://mirrors.cloud.tencent.com/npm/ npm run redeploy
```

- Frontend: http://localhost:5173
- Backend: http://localhost:3001

## Test

```bash
npm test
```
